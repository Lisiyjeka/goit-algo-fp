## Висновки до симуляції кидків двох кубиків методом Монте-Карло

### Методологія дослідження
Для дослідження ймовірностей сум при киданні двох кубиків було використано метод Монте-Карло - статистичний метод моделювання, який базується на великій кількості випадкових експериментів. Симуляція включала 100,000 кидків двох шестигранних кубиків.

### Теоретичні основи
При киданні двох кубиків можливі суми значень знаходяться в діапазоні від 2 до 12. Теоретичні ймовірності розраховуються на основі комбінаторики

## Ключові висновки

1. Валідація методу Монте-Карло
Результати експерименту підтверджують коректність і точність методу Монте-Карло для обчислення ймовірностей. При 100,000 ітераціях отримані результати практично ідентичні аналітичним розрахункам.

2. Вплив кількості ітерацій
***Точність методу залежить від кількості симуляцій:***

100,000 кидків: середня похибка 0.07%

Чим більше ітерацій, тим вища точність

3. Розподіл ймовірностей
***Результати підтверджують теоретичний розподіл:***

Нормалізація: сума всіх ймовірностей = 100%

Біноміальний розподіл: характерний для суми двох незалежних випадкових величин

4. Практична значущість
***Метод Монте-Карло ефективний для:***

Складних ймовірнісних систем, де аналітичний розрахунок складний

Перевірки теоретичних моделей

Моделювання реальних процесів з випадковими факторами



## Загальний висновок
Експериментальні результати повністю підтверджують теоретичні розрахунки. Метод Монте-Карло продемонстрував високу точність (99.93%) при моделюванні 100,000 кидків двох кубиків. Це підтверджує ефективність даного методу для розв'язання ймовірнісних задач та його практичну придатність для статистичного моделювання.
